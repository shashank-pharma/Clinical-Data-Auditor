# Clinical-Data-Auditor

A Python-based clinical trial participant auditing system built using Object-Oriented Programming (OOP), Inheritance, Custom Exceptions, and Logging.

## Features

* Validates participant eligibility based on age criteria.
* Verifies diagnosis integrity for trial enrollment.
* Uses custom exceptions for error handling.
* Generates audit logs for tracking validation results.

## Concepts Used

* Classes & Objects
* Inheritance
* Custom Exceptions
* Exception Handling
* Logging

## Files

* `Clinical_Data_Auditor.ipynb` – Jupyter Notebook
* `Clinical_Data_Auditor.py` – Python Script
* `trial_audit.txt` – Audit Log File

## Learning Outcome

This project demonstrates how clinical trial participant data can be validated using Python while applying OOP principles, exception handling, and audit trail generation.
